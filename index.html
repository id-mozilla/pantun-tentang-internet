Petunjuk kontribusi
  1. Fork repository ini : Pantun Tentang Internet
  2. Clone HASIL fork repository tersebut
  3. Buat branch baru `git branch nama-branch`
  4. Pindah ke branch branch baru `git checkout nama-branch`
  5. Buat folder dengan judul pantun, dengan format huruf kecil dan tanpa spasi, misal "judul-pantun"
  6. Buat file index.html, lalu tuangkan pantun kalian, tidak perlu berpenampilan menarik, yang penting kontennya.
  7. Add file
  8. Commit
  9. Push
  10. Buat Pull Request
