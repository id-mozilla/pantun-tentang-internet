Petunjuk kontribusi
  1. Fork repository ini : Pantun Tentang Internet
  2. Clone HASIL fork repository tersebut
  3. Buat folder dengan judul pantun, dengan format huruf kecil dan tanpa spasi, misal "judul-pantun"
  4. Buat file index.html, lalu tuangkan pantun kalian, tidak perlu berpenampilan menarik, yang penting kontennya.
  5. Add file
  6. Commit
  7. Push
  8. Buat Pull Request
